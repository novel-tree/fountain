# Fountain

A dedicated platform to introduce novelists and their works to the world.

## Mission

Fountain is a non-commercial project created with a single goal: **to increase the reading population, especially in literature.** 

We believe that by providing a beautiful, accessible space to discover both classic and contemporary authors, we can help more people find their next favorite book. Our app features images, quotes, and direct links to the publishers and social media (SNS) of living authors to bridge the gap between creators and readers.

## Non-Commercial & Open Content

- **Purpose:** This app does not earn money. It is purely for the promotion of literature.
- **Content Updates:** We welcome GitHub issues from publishers and authors to update or correct their content.
- **Repo-Stored Content:** All data and content displayed in the app are stored directly within this repository to ensure transparency and community-driven maintenance.

## Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (App Router)
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/) (or Vanilla CSS as per project progress)
- **Deployment:** [Vercel](https://vercel.com/)

## Getting Started

First, install dependencies:

```bash
pnpm install
```

Then, run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## How to Contribute

### For Authors and Publishers
If you are an author or a publisher and would like to:
- Update your profile image or quotes.
- Correct or add links to your SNS or purchase pages.
- Request the removal of specific content.

Please [open a Content Update issue](https://github.com/michaelwilkins/fountain/issues/new?template=content-update.yml). We prioritize these requests to ensure accuracy and respect for your work.

### For the Community
- **Suggest an Author:** If there's a novelist you think should be featured, let us know via a [New Author Suggestion](https://github.com/michaelwilkins/fountain/issues/new?template=new-author.yml).
- **Bug Reports:** Found a technical glitch? [Report it here](https://github.com/michaelwilkins/fountain/issues/new?template=bug-report.yml).
