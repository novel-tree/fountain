# Security Policy

## Supported Versions

Only the latest version of Fountain is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| latest  | :white_check_mark: |

## Reporting a Vulnerability

We take the security of Fountain seriously. If you believe you have found a security vulnerability, please report it to us by following these steps:

1. **Email us:** Send a detailed report to [INSERT EMAIL ADDRESS].
2. **Include details:** Describe the vulnerability, how it can be exploited, and its potential impact.
3. **Wait for response:** We will acknowledge your report within 48 hours and provide a timeline for a fix.

Please do not open a public GitHub issue for security vulnerabilities. We will coordinate a public disclosure once the issue is resolved.
